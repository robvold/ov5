# Ov5

This is a cpp code runner
## Prerequisites:
node.js and npm  
docker  
java
## Installation:
Server - in a terminal:
```bash
cd ov5LarsSergioRobin/ov5server
docker build -t ov .
java Server
```
Client - in a new terminal:
```bash
cd ov5LarsSergioRobin/ov5frontend
npm install
npm start
```
Your app is now running on localhost:3000
